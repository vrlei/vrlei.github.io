AR/VR行业疯狂“吸金”！2017年总融资额达30亿美元|苹果|VR|AR技术_新浪科技_新浪网

    【猎云网】1月9日报道 （编译：大黄）



根据分析公司Digi-Capital的数据，在2017年，AR和VR产业的初创公司筹集到总计30亿美元融资。该公司认为，尽管环绕AR/VR产业的声势依然减弱，资本依然在涌入这个领域。



尽管该报告显示2017年流入行业的资金相较2016年实现了增长，总体交易量却较上年有所下降，其中四笔交易提供了超过半数融资。



Magic Leap =$5.02亿美元



Improbable = $5.02亿美元



Unity = $4.00亿美元



Niantic = $2.00 亿美元



Niantic、Improbable和Unity这样的业内巨子可以凭借定将改变行业未来趋势的企划从投资者手中募集到上亿美元融资。他们强势的游戏背景毫无疑问是拿到这些融资的关键。







Magic Leap是行业内最大的黑马。对于他们将发布的产品，我们有一个大概的理解：这款产品是面向企业的。我们没有关于发布日期的零售价格的信息。更重要的是，我们不知道Magic Leap会如何平衡面向公司的产品和面向一般消费者的产品。



对于在2016和2017年开启种子轮的小公司而言，交易流量变得越来越小了（Crunchbase有一篇对此的研究）显示出泡沫的存在，更多的AR/VR初创公司得到种子轮，更少的公司活过这一轮。



在2017年下半夜，备受瞩目的Apple ARKit和Google ARCore这两款开发工具使得关注焦点从VR头戴设备转向AR移动设备。由于构筑于此的app太偏向视觉效果并且不能提供良好的回报，市场对此反应平平。预计在未来的十年中，消费AR产品在等待Apple、Microsoft和Magic Leap为市场指明方向前不会有什么大动作



未来依然是充满希望的。尽管2017年市场自我纠正AR/VR产业的泡沫使得部分资金转向。Google, Apple, Facebook 和 Microsoft依然拥有足够的资本资助真正有价值的公司。我们很高兴能看到大量资本被注入初创公司。但对于新兴技术行业，看到交易量大幅减少令人担忧。
