    第1页CES 2018:联想发力虚拟设备 分别推出VR、AR、MR设备

    　　在一年一度的科技盛会CES上，汇集了来自世界各地的大牌厂商。其中，国内PC大厂也参加了本次大展。

    　　从联想展出产品来看，其本次主打方向之一就是虚拟技术方面。在本次CES 2018上，联想推出了多款虚拟设备，分别涵盖了虚拟现实、增强现实和混合现实三大领域。

    CES 2018:联想发力虚拟设备 分别推出VR、AR、MR设备

    　　虚拟现实方面，联想推出了全球首款支持Daydream标准的VR一体机Mirage Solo。该一体机搭载了高通骁龙835移动平台，采用分辨率为2560 x 1440 LCD液晶显示屏，提供110度视角，拥有4GB+64GB存储空间。

    　　同时，联想还展示了一款搭配VR一体机使用的360度全景视频相机Mirage Camera。该相机配备1300万像素鱼眼双摄像头，可拍摄4K级别质量的2D图像、立体图像及3D全景视频。

    CES 2018:联想发力虚拟设备 分别推出VR、AR、MR设备

    　　增强现实方面，联想推出新一代的AR眼镜New Glass C220与C300，同以往一样，这两款AR设备同样是主打B端市场。该AR眼镜可提供单眼20°的FOV，保证视野同时也能有效传递数字信息。配备的坚固且佩戴舒适护目镜以及800mAh大电池，适合工业、医疗等不同应用场景下长时间作业。

    CES 2018:联想发力虚拟设备 分别推出VR、AR、MR设备

    　　最后是混合现实方面，联想推再次展示了自家的Windows Mixed Reality头显设备——Explorer。该头显采用微软HoloLens的技术，搭载1440×1440的OLED显示面板，提供6自由度的内向外追踪，这意味着用户无需使用任何外部跟踪装置或摄像头的空间。
