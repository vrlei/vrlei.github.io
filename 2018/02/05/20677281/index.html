    　　原标题：高通携手歌尔发布骁龙VR820参考设计平台

    　　在9月2日的IFA展上，歌尔有限公司(以下简称“歌尔”)正式对外宣布，高通公司联手歌尔发布了基于骁龙820芯片的虚拟现实参考设计平台—–VR820。该平台融合了骁龙820芯片强大的计算能力和歌尔业界领先的VR整机设计和制造经验，为开发者提供了一套开创性的高性能VR一体机开发环境。

    　　在本次与高通强强联合中，歌尔提供了创新的系统集成设计，包括新颖炫酷、充分考量人体工学的机器人型外观与结构设计，大幅提升了散热效率的一体机专属冲压片散热设计，基于歌尔在声学部分强大优势的前沿声学设计，自主研发设计fov高达100大视场角镜片提供领先的光学设计，以及先进的电路设计和固件开发;高通的芯片组则为系统提供了强大的计算、图像处理、传感器集成、系统连接和电源管理功能。现场发布的这款骁龙VR820虚拟现实头盔拥有卓越的芯片处理能力：专属打造的 64 位 Qualcomm？ Kryo？ 四核 CPU 提供 性能计算，Hexagon 680 DSP 实现了高质量的视频、 频处理，基于All-Ways Aware？的实现超低功耗;拥有领先的FOV100度高清晰非球面光学镜头，单眼达到1440 x 1440分辨率、刷新频率70Hz的显示屏，采用最新Qualcomm？ Adreno？ 530 GPU提升最多40%图形和计算性能，结合超高速传感、包括陀螺仪，加速度计，地磁传感器，将延迟处理控制在18ms以下，诠释全新的VR视觉效果;同时，集成了基于双摄像头的眼球追踪技术、双前置摄像头的六自由度(6DOF)跟踪技术，以及四麦克风主动降噪技术，打造流畅自然的交互体验。

    　　“这是一个激动人心的时刻!”歌尔总裁姜龙表示：“VR820平台的首次成功发布，代表着虚拟现实技术在商业化的道路上得到了主流芯片厂商的强力支持。歌尔凭借在VR领域的多年经验积累，与高通公司进行深度合作，完成了VR820的整机设计，为开发者提供了高性能的硬件、算法和整机的参考设计。”

    　　高通副总裁Anthony Murry也对媒体表示：“通过对关键技术和功能的整合，骁龙VR820能切实满足移动虚拟现实产品的需求，为OEM厂商提供了快速开发产品，进行大量生产的可能性。”

    　　平台的发布代表着移动VR技术的进步和走向成熟的开始，歌尔经过多年持续的耕耘和布局，形成了完整的VR研发和制造优势，包括声学算法、图像处理、光学镜头设计与制造、关键零件制造、整机仿真与测试以及基于Android的VR相关软件的开发等。
